# 🌍 Wander Diaries

A simple travel blog web application where users can read and share travel experiences from around the world. Users can browse travel stories, submit their own adventures, and stay updated with the latest travel news through a clean and responsive interface.

---

## ✨ Features

* 🏠 Home page with a travel-themed design
* 📖 Read travel stories shared by users
* ✍️ Upload your own travel experience
* 📍 Add a travel title, description, date, and location
* 📰 View the latest travel news
* 🔒 User input is sanitized using **sanitize-html**
* 📱 Responsive design for different screen sizes
* 🚫 Custom 404 error page

---

* 🌐 HTML5

## 🚀 Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone <repository-url>
```

### 2️⃣ Navigate to the project folder

```bash
cd Blogg_APP
```

### 3️⃣ Install project dependencies

```bash
npm install
```

### 4️⃣ Install sanitize-html (if not installed automatically)

```bash
npm install sanitize-html
```

### 5️⃣ Start the application

Using Nodemon:

```bash
npm run dev
```

or using Node.js:

```bash
node server.js
```

### 6️⃣ Open your browser

```
http://localhost:8000
```

*(Use the port configured in `server.js`.)*
